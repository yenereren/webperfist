# Webperf.ist

Web performans üzerine odaklanmış bir topluluktur.Amacımız ulaştığımız kitlelere, web'i olabildiğince hızlı ve doğru bir şekilde ulaştırmaktır.

## Install

gatsby 3 and node-sass 5 requires node v12 or higher

```
npm install
```

```
npm run develop
```

```
npm run build
```
